# Ganty Design & Decoration

Static portfolio website for Ganty Design & Decoration W.L.L.

## Local Preview

Serve the folder with any static web server, then open `index.html`.

```powershell
npx --yes http-server . -a 127.0.0.1 -p 5173
```

## Pages

- `index.html` - homepage
- `projects.html` - project listing
- `projects/*.html` - project detail pages
